## How to run

- run app
  - create `.env` file with configuration, e.g.

  ```properties
  DB_USER=test
  DB_PASS=test
  DB_DATA_PATH=~/h2-data
  ```

  - run `docker-compose pull & docker-compose up`
- open start page on `localhost`
