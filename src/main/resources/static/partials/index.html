ИП Осипова

Рекомендуются следующие действия:

  1. Добавление подарков
    Нажать на кнопку "Добавление подарка". Отобрать содержимое подарка, указать цену и, если необходимо, название подарка.
  2. Просмотр готовых подарков
    При нажатии на кнопку "Список подарков" отображаются все добавленные ранее подарки.
  3. Распечатать подарок
    Если необходимо произвести распечатку подарку, то необходимо из списка подарков выбрать нужный и нажать кнопку "Просмотр". На появившейся странице отображается вся доступная информация о подарке. Для печати необходимо нажать на кнопку "Скачать" => Файл => Печать.
